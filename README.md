# UAS-PROG-KelompokVII
Sistem Analitik Smart Sustainability Campus UNESA berbasis C++ dan Python. Proyek ini mengolah 29.200 data sensor (CO₂, energi, air, sampah) dari 20 gedung selama setahun untuk menghasilkan Sustainability Rating: Green, Warning, dan Critical guna mendukung operasional kampus ramah lingkungan.
