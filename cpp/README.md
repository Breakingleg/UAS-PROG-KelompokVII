# Modul Analitik C++ - Smart Sustainability Campus

Modul ini berisi implementasi sistem pemrosesan data sensor IoT menggunakan bahasa C++. Pemrosesan data dilakukan dengan memanfaatkan struktur data ketat seperti `std::map`, `std::vector`, dan `std::set` untuk menjamin efisiensi memori saat mengolah puluhan ribu baris record.

## Komponen File
- `main.cpp`: Kode sumber utama C++ yang menangani parsing CSV, kalkulasi statistik dasar, akumulasi konsumsi per gedung, dan penentuan peringkat keberlanjutan kampus menggunakan pendekatan `.front()`.

## Prasyarat Sistem
Pastikan perangkat sudah terinstal compiler GCC/G++ yang mendukung standar C++11 atau versi di atasnya.
